# Rooftop Runner
## Pomozi čikici da preskoči sve zgrade!
_Obrati pažnju na to da zgrade nisu podjednako udaljene međusobno!_


## Kontrole

| Taster        | Funkcija            |
| :-----------: |:-------------:      |
| G             | Pokretanje igre     |
| S             | Zaustavljanje igre  |
| R             | Vraćanje na početak |
| J             | Kratak skok         |
| K             | Dugačak skok        |
| ESC           | Izlazak iz igre     |
